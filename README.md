# 🔥 Tuffetch
TuffFetch is a lightweight system tool that displays information from your computer directly in the terminal, clearly and colorfully.

## 💫 Compatibility
- *macOS*
- *GNU/Linux*

## 🚀 Dependency
- Python3
    - Psutil
    - Rich

## 🌴 Installation

Clone the repo :
```bash
git clone https://github.com/fluff2513/tuffetch.git
cd tuffetch
```

Install the dependency:

```bash
pip3 install psutil rich
```

Run Tuffetch :

```bash
python3 tuffetch.py
```
